Recommendable (web) applications

## Chat / Messaging

- **[once-campfire](https://github.com/basecamp/once-campfire)** - GitHub repository by Basecamp
- **[Revolt](https://revolt.chat/)** - A chat application designed with community in mind
- **[Gotify](https://gotify.net/)** - A simple server for sending and receiving messages

## CMS

- **[Ghost CMS](/posts/ghost-cms/)** - An open-source CMS favored for content sites
- **[Writebook](https://once.com/writebook)** - Publish your own books on the web for free

## Development

- **[TestLink](https://github.com/TestLinkOpenSourceTRMS/testlink-code)** - Open source test and requirement management system

## Document Management

- **[paperless-ngx](https://github.com/paperless-ngx/paperless-ngx)** - A document management system for scanning, indexing, and archiving documents

## Home

- **[Home Assistant](https://www.home-assistant.io/)** - Open source home automation with local control and privacy first, ideal for Raspberry Pi or local servers

## Hosting

- **[Databasus](https://databasus.com/)** - Free tool for PostgreSQL scheduled backups with cloud storage and notifications
- **[Dokku](/posts/dokku-open-source-heroku-alternative/)** - Self-hosted open-source PaaS alternative to Heroku
- **[copyparty](https://github.com/9001/copyparty)** - Portable file server with WebDAV, FTP, TFTP, and media indexing

## Mail

- **[Plunk](https://www.useplunk.com/)** - Open-source email platform combining marketing, transactional, and broadcast emails

## Project Management

- **[OpenProject](https://www.openproject.org/)** - Open source project management with task management, Gantt charts, boards, and team collaboration

## Tasks

- **[TaskTrove](https://tasktrove.io/)** - Self-hostable, privacy-focused TODO list management application
